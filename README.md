## ðŸš€ Quick start

1.  **Install all composer dependencies**

    ```sh
    composer install
    ```

2.  **Install all npm dependencies**

    ```sh
    npm install
    ```

3.  **Migrate the database**

    ```sh
    php artisan migrate
    ```

4. **Make sure there is a symbolic link**

    ```sh
    php artisan storage:link
    ```

5. **Compile assets**

   ```sh
    npm run dev
    ```
   
7. **Start the development server **

   ```sh
    php artisan serve
   ```
